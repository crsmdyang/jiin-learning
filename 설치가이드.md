# 🏰 지인이의 배움 나라 — 설치 가이드

전체 과정은 약 15분이면 끝납니다. 크게 3단계예요:
**① Firebase 만들기(로그인 서버) → ② 설정값 붙여넣기 → ③ GitHub Pages에 올리기(무료 호스팅)**

> ①~③을 하기 전에도 폴더의 `index.html`을 열면 "게스트 모드"로 모든 게임이 동작합니다.
> Firebase는 로그인·기기 간 진도 동기화를 위한 것입니다.

---

## ① Firebase 프로젝트 만들기 (약 5분, 무료)

1. https://console.firebase.google.com 접속 → 구글 계정으로 로그인
2. **프로젝트 추가** 클릭
   - 이름: `jiin-learning` (아무거나 가능)
   - Google 애널리틱스: **사용 안함** (꺼도 됨) → **프로젝트 만들기**
3. 왼쪽 메뉴 **빌드 → Authentication** → **시작하기**
   - **이메일/비밀번호** 클릭 → **사용 설정** 켜기 → 저장
4. 왼쪽 메뉴 **빌드 → Firestore Database** → **데이터베이스 만들기**
   - 위치: `asia-northeast3 (서울)` → **프로덕션 모드** → 만들기
   - 만들어지면 **규칙(Rules)** 탭 클릭 → 내용을 전부 지우고 폴더 안의
     `firestore.rules` 파일 내용을 붙여넣기 → **게시**
5. 왼쪽 위 **⚙(프로젝트 설정) → 일반** 탭 → 아래 "내 앱" → **`</>` (웹)** 아이콘 클릭
   - 앱 닉네임: `portal` → 앱 등록 (호스팅 체크는 안 해도 됨)
   - 화면에 나오는 `firebaseConfig = { apiKey: "...", ... }` 부분을 **복사**

## ② 설정값 붙여넣기

- 폴더의 `js/firebase-config.js` 파일을 메모장으로 열고,
  `PASTE_...` 로 되어 있는 부분을 ①-5에서 복사한 값으로 바꿔 저장하세요.
- 또는 복사한 firebaseConfig를 Claude에게 보내주시면 제가 넣어서 다시 드릴게요.
  (이 값은 공개되어도 되는 클라이언트 설정값입니다. 비밀번호가 아니에요.)

## ③ GitHub Pages에 올리기 (약 5분, 무료)

1. https://github.com 로그인 → 오른쪽 위 **+ → New repository**
   - 이름: `jiin-learning` / **Public** → **Create repository**
2. 만들어진 페이지에서 **"uploading an existing file"** 링크 클릭
   - 압축을 푼 **word-portal 폴더 안의 내용물 전체**(index.html, js, data, wordkingdom 폴더 등)를
     드래그해서 업로드 → **Commit changes**
   - ⚠️ word-portal 폴더 자체가 아니라 그 **안의 파일들**을 올려야 합니다
     (index.html이 저장소 맨 위에 보여야 해요)
3. 저장소 **Settings → Pages** (왼쪽 메뉴 맨 아래쪽)
   - Branch: `main`, 폴더 `/ (root)` → **Save**
4. 1~2분 후 `https://아이디.github.io/jiin-learning/` 에서 접속됩니다!

## ④ 마지막 연결 (1분)

- Firebase 콘솔 → **Authentication → 설정 → 승인된 도메인** → **도메인 추가**
  → `아이디.github.io` 입력 → 추가
  (이걸 해야 GitHub 주소에서 로그인이 됩니다)

## ⑤ 갤럭시패드에서 쓰기

1. Chrome에서 `https://아이디.github.io/jiin-learning/` 접속
2. 메뉴(⋮) → **홈 화면에 추가** → 앱처럼 아이콘으로 실행!
3. 가입(부모 이메일) → 아이 프로필 만들기 → 끝!

## 자주 묻는 것

**Q. 무료인가요?**
Firebase Spark 요금제(무료)로 충분합니다: 로그인 무제한, DB 1GB·일 5만 회 읽기.
아이 몇 명이 쓰는 수준에서는 절대 초과하지 않아요. GitHub Pages도 무료입니다.

**Q. 새 프로그램은 어떻게 추가하나요?**
`index.html`의 `PROGRAMS` 목록에 한 줄 추가하고, 새 폴더(예: `mathland/`)를 만들어
올리면 됩니다. Claude에게 "수학 나라 만들어줘"라고 하시면 같은 구조로 만들어 드려요.

**Q. 진도가 안 넘어가요 / 로그인이 안 돼요**
- ④ 승인된 도메인 추가를 했는지 확인
- `js/firebase-config.js`에 값을 제대로 붙여넣었는지 확인

**Q. 예전(파일 버전) 진도를 옮길 수 있나요?**
기존 HTML 파일의 부모 설정 → 백업 내려받기로 JSON을 저장한 뒤,
새 사이트 게임의 부모 설정 → 백업 불러오기 하면 됩니다.
